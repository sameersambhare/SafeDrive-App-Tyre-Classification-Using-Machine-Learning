## 🔧 Customization Guide

### Change App Colors
Edit `src/styles/theme.ts`:
```typescript
export const colors = {
  primary: { main: '#YOUR_COLOR' }
}
```

### Change API Endpoint
Edit `src/utils/api.ts`:
```typescript
const API_BASE_URL = 'https://your-api.com/v1';
```

### Add New Screen
1. Create `src/screens/YourScreen.tsx`
2. Import in `src/App.tsx`
3. Add to navigation

### Modify Existing Screen
Edit files in `src/screens/` - hot reload works!

### Common Issues

**"npm command not found"**
- Install Node.js: https://nodejs.org/

**"Can't connect to camera"**
- Test on physical device (emulator limited)
- Check permissions in Android settings

**"API connection failing"**
- Check API endpoint in `src/utils/api.ts`
- Verify backend is running
- Check network connectivity

**"App keeps crashing"**
- Check console for error message
- Clear cache: `npm start -c`
- Reinstall: `rm -rf node_modules && npm install`
